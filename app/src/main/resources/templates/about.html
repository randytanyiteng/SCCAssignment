⭐ About Cloud Favorites

A simple and powerful bookmarking web application

📌 Project Overview

Cloud Favorites is an open-source bookmarking platform built with Spring Boot 2.x. Users can save, categorize, and search their favorite websites and articles online.

🚀 Tech Stack

Spring Boot 2.x Thymeleaf MySQL Bootstrap 3 jQuery Maven Docker
